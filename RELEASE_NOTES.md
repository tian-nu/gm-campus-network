# 校园网自动认证工具 v1.0

## 发布说明

### 新增功能

1. **网络变化监听**
   - 自动检测有线/无线网络切换
   - 检测IP地址变化
   - 检测系统休眠/唤醒
   - 网络变化时自动重新认证

2. **智能重连机制**
   - 断线自动重连
   - 账号封禁检测与自动重试
   - 指数退避策略避免频繁请求
   - 动态调整检测间隔

3. **系统托盘支持**
   - 最小化到系统托盘
   - 托盘状态显示（在线/离线/重连中）
   - 托盘通知提醒

4. **代理/VPN检测**
   - 登录前检测代理状态
   - 严格模式阻止代理登录
   - 防止账号因代理被封禁

### 修复的问题

- 修复登录按钮卡住问题
- 修复重连失败时UI状态未更新问题
- 优化网络检测逻辑
- 添加异常处理机制

### 系统要求

- Windows 10/11
- 无需安装Python环境
- 首次启动可能需要几秒钟

### 使用方法

1. 下载 `校园网自动认证工具.exe`
2. 双击运行程序
3. 输入学号和密码
4. 点击"一键登录"
5. 可选择"启动时自动登录"和"最小化到托盘"

### 注意事项

- 如果被杀毒软件拦截，请添加信任
- 建议关闭代理/VPN后再登录校园网
- 账号被封禁时会自动等待并重试
